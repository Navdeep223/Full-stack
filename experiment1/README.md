# Experiment 1: Web Application Components

## Aim
To design and develop different web application components using HTML, CSS, and JavaScript.

## Description
This experiment consists of multiple parts that demonstrate the use of frontend web development concepts. Each part represents a different webpage with specific functionality and design.

## Parts Included

### 1. Form
-Description: A web form designed to collect user input using various HTML form elements.

### 2. Bank Interface
- Description: A simple banking interface demonstrating layout design and basic user interaction.

### 3. Dashboard
- Description: A dashboard page used to display structured information in a user friendly format.

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## Output
All the above webpages open successfully in the browser and demonstrate the required functionality.

This experiment helped in understanding webpage structure, form elements, layout design, and basic interactivity using JavaScript.
