# React Experiments – Frontend Development

This project contains multiple React experiments demonstrating core frontend and JavaScript concepts including component-based architecture, state management, routing, and object-oriented programming.

---

# 📘 Experiment 3.1 – ProductCard Component

## 🎯 Aim
To create a reusable ProductCard component using React props.

## 📌 Features
- Reusable React component
- Props for dynamic data
- Conditional rendering for stock status
- Styled using CSS
- Responsive card layout

## 🛠 Concepts Used
- Functional Components
- Props
- Conditional Rendering
- CSS Styling

---

# 📗 Experiment 3.2 – Library Management UI

## 🎯 Aim
To build a library management interface with add, search, and remove functionality.

## 📌 Features
- Display book list
- Search functionality
- Add new books
- Remove books
- Form handling
- State management using hooks

## 🛠 Concepts Used
- useState Hook
- CRUD operations
- Form handling
- Dynamic rendering

---

# 📙 Experiment 3.3 – Person Class Hierarchy

## 🎯 Aim
To demonstrate OOP concepts in JavaScript using class hierarchy.

## 📌 Features
- Base Person class
- Student and Teacher subclasses
- Method overriding
- Polymorphism
- Rendering class data in UI

## 🛠 Concepts Used
- ES6 Classes
- Inheritance
- Polymorphism
- Method Overriding
